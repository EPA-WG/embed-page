Storage insulation <embed-page/> demo, Embeddable Progressive Applications Working Group

Manual test 
IFRAME
    1. page-storage.js will set localStorage 'a' to localStorage + url 
    2. GET the localStorage, expected localStorage page-storage.html?src=iframe
        as script from IFRAME will be executed after this page one.
    3. GET the sessionStorage, expected sessionStorage page-storage.html?src=iframe
    4. SET own key/value in each storage on page and get in IFRAME observing the value from page inside of IFRAME.
    5. observe same key value in embed-page context is NOT propagated from the host page.

localStorage - embed-page
    1. page-storage.js will set localStorage 'a' to window.location+" localStorage
    2. Get the 'a' key on this page. Value's location part should match storage.html
    3. same within embed-page. Value's location part should match page-storage.html

sessionStorage - embed-page
    1. page-storage.js will set sessionStorage 'a' to window.location+" sessionStorage
    2. Get the 'a' key on this page. Value's location part should match storage.html
    3. same within embed-page. Value's location part should match page-storage.html

Scope check
    for localStorage & sessionStorage
    1. set key/value
    2. close and reopen browser tab
    3. GET key, observe the previously set value is NOT set when scope set to "instance" and shown in other scopes.

Cookies
    1. set cookies a=b, set cookies c=d
    2. get cookie should have both key/values presented
    3. refresh page, get cookies should have both key/values presented
